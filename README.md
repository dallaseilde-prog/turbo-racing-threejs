# Turbo Racing Day

A high-fidelity 3D racing game built with Three.js.

## Features
- 3 Tracks: Sunny Hills, Desert, Night City.
- 3 Cars: Red Fire, Blue Jet, Shadow.
- Realistic physics and collision detection.
- Lap timer with personal best tracking.
- Nano Banana Pro High-Fidelity assets.

## How to Run
1. Open `index.html` in a modern web browser.
2. Ensure you have an internet connection to load Three.js from CDN, or run `download_deps.py` to get it locally.
